## ðŸ’» Installation

### Prerequisites
-   **GPU**: NVIDIA RTX 3090/4090/5090 (24GB+ VRAM recommended).
-   **OS**: Windows 11 + WSL2 (Ubuntu 22.04).
-   **Storage**: 100GB+ on fast SSD (L: Drive recommended).

### Setup
1.  **Clone Repository**:
    ```powershell
    git clone https://github.com/YoneRai12/ORA.git
    cd ORA
    ```
2.  **Install Environment**:
    ```powershell
    setup_env.bat
    ```
3.  **Launch**:
    -   **Right-Click Desktop -> "Start ORA Bot"**
    -   Or run `start_services.bat`.
